Sharing images in from the web

If this app is installed to your home screen (see Info tab for install steps), you can use Android's regular Share menu: on any webpage, tap an image (long-press to bring up image options, or use the page's Share button), choose Share, and pick Wardrobe from the list. It'll open here ready to add as a new item.

This relies on a browser feature that isn't available on every Android browser. If it doesn't show up as a share option, use the bookmarklet fallback below instead — note that on some mobile browsers (notably Chrome and Opera on Android) bookmarklets are blocked entirely and this fallback won't work either; in that case, save images to your gallery and upload them normally.

Installing this app (for Android sharing)

To use Android's Share menu to send images here, this page needs to be installed to your home screen as an app (not just bookmarked). In Chrome: tap the ⋮ menu → Add to Home screen → Install. Once installed, open it at least once from the home screen icon before trying to share into it.

This file

This is a self-contained wardrobe catalog. All your items, tags, outfit boards, and images are stored inside this single HTML file — nothing is sent anywhere. To back up or sync, use Save / Export to download a fresh copy (named with a timestamp), and put it wherever you sync from (e.g. a OneDrive folder). Use Open file… to load a previously saved copy.

Because everything lives in the file itself, if two people edit separate copies and save separately, you'll end up with two files that have diverged — there's no automatic merge yet. Compare the stats above between copies if something seems off.
